Food Delivery Website

A modern, responsive food ordering website inspired by Swiggy. Built using React, it features live restaurant listings, infinite scroll, search/filter, cart functionality, error boundary handling, shimmer loading UI, and light/dark theme toggle. Redux is used for global state management. Data is fetched from a Swiggy-like mock API, and the project is optimized for performance using Vite.

Tech Stack

  • React.js
  • Redux Toolkit
  • Tailwind CSS
  • Vite

Features

  • Real-time restaurant listings
  • Cart & Checkout functionality
  • Search & Filters with infinite scroll
  • Light/Dark mode toggle
  • Error handling & shimmer loading UI
