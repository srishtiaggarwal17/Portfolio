Job Portal Website

This is a full-stack MERN job portal designed for students and recruiters. It features user authentication, job creation, application management, and seamless user experience. The backend is built with Node.js and Express, the frontend with React, and MongoDB as the database.

Tech Stack

  • MongoDB
  • Express.js
  • React.js
  • Node.js

Features

  • Student & Recruiter login
  • Job listings & applications
  • CRUD operations
  • Real-time updates
