Book N Travel Website

A responsive travel booking website that helps users explore and book travel destinations with ease. Developed using React and Tailwind CSS, this project features a sleek landing page, animated scroll effects, dropdown navigation, and mobile responsiveness. Routing is managed via React Router, and error boundaries ensure graceful fallback UI for broken or invalid routes.

Tech Stack

  • React.js
  • React Router
  • Tailwind CSS
  • JavaScript

Features

  • Animated scroll effects with pop-ups
  • Responsive design for all devices
  • Interactive navbar with dropdowns
  • Custom 404 error page using useRouteError
